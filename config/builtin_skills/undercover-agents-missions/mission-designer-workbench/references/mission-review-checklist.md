# Mission Review Checklist

- Confirm the mission starts from clear inputs and ends with an intentional output.
- Confirm every branch that should continue has an explicit continuation.
- Confirm loops return to their own body and that once-per-run work lives after the loop finishes.
- Confirm node names are readable and describe the job each step performs.
- Confirm custom code is only used where a purpose-built node would not be enough.
- Confirm outputs, selected variables, or final response fields match what the user actually needs.
