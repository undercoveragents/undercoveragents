# How Agents Use Skills

Use this skill for questions about what happens after a catalog is attached to an agent.

## Progressive disclosure

- Attached skills are available to the agent as a menu of deeper manuals.
- The agent chooses a relevant skill when the question needs more than its base instructions.
- This keeps the base prompt compact while preserving access to detailed guidance.

## Skill resources

- Some skills include bundled reference files for checklists, examples, or deeper explanations.
- The agent can read those resources only when the skill alone is not enough.

## Practical guidance

- If an agent gives shallow answers, it may need a stronger skill or a better catalog assignment.
- If an answer is too generic, check whether the correct area skill was activated.
- Use skills for better reasoning and explanation, not for hiding required product configuration.
