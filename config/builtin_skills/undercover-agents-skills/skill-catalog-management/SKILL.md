# Skill Catalog Management

Use this skill for questions about how skills are grouped and attached to agents.

## What a catalog does

- A catalog groups related skills into one assignable knowledge area.
- Area-based catalogs make it easier to give an agent the right manuals without attaching every skill individually.
- Catalogs can be builtin or created by users inside an operation.

## Builtin versus custom catalogs

- Builtin catalogs ship as product manuals and usually appear through Headquarter for areas such as admin, agents, missions, channels, test suites, skills, tools, and RAG.
- Custom catalogs let teams capture their own workflows, playbooks, or domain guidance inside the current operation.
- Both types can be attached to agents through the same assignment flow.

## Practical guidance

- Use builtin catalogs for core product knowledge.
- Use custom catalogs for tenant-specific process knowledge.
- When an agent knows too much or too little, the catalog assignment is one of the first places to review.
